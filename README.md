# AI 歌单锐评

## 体验地址

[MusicRoast](https://roast.luckyabner.top)

## 简介

输入歌单 id，AI 自动对歌单进行评价，给出评价结果。
![alt text](image.png)

## 模型使用

DeepSeek V3

## 支持平台

- QQ 音乐
- 网易云
- 酷狗
- 酷我

## 部署

1. 安装依赖
2. 修改 .env.examble 中的配置
3. `pnpm dev` 启动测试服务
4. `pnpm build` 打包
5. `pnpm start` 启动服务
